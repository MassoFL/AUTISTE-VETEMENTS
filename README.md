# Autiste Vêtements

Boutique en ligne minimaliste avec interface TikTok-style.

## Setup

1. Créer un projet Supabase
2. Exécuter le SQL dans `supabase-schema.sql`
3. Créer un bucket Storage nommé `images` (public)
4. Mettre à jour les credentials dans `config.js`

## Déploiement Vercel

1. Push le code sur GitHub
2. Importer le projet dans Vercel
3. Déployer

## Structure

- `index.html` - Site principal
- `admin.html` - Panel admin
- `story.html` - Page "The Story"
- `config.js` - Configuration Supabase

## Fonctionnalités

- Upload d'images
- Gestion des couleurs
- Posts avec multiple images
- Recherche
- Interface TikTok-style
